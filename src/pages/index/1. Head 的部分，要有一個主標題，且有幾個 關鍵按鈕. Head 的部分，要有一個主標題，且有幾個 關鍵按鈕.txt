1. Head 的部分，要有一個主標題，且有幾個 關鍵按鈕

    A. 帳號切換，當沒有手動設置帳號的時候，用預設帳號

    B. 複製當前 json 配置，用來輸出 / 從剪貼簿讀取 json 配置

    C. 增加組項目按鈕，其中要有幾個預設配置 + 空白模板

2. [組配置]，每個組都要折疊 收縮，且 可以勾選是否應用 該組數據，且可以讓用戶設置每個組的標題，以及一行說明，且每個組配置都有一個刪除按鈕

[組配置]中，有子項目配置，每個子項目前面 最前面是 圖片預設40x40 hover 不影響布局的情況 顯示 為 80x80 ，然後是 input 按鈕從 public/locales/<語言>/material.json 讀取 列表接著是數量按鈕，然後每列都要有 列備註，且每列最後都要有刪除按鈕

3. [小節] 使用以上數據，應用 計算每個材料的 SUM，且要有2個按鈕，1個是複製小節為 json 一個是複製為 CSV

4. 數據，自動儲存到 localStorage
